# CNI-Meeting-Themes-Topics-2015-2019
A prototype visualization of CNI meeting themes, topics, and presentations from 2015-2019 Spring and Fall member meetings

This sunburst chart presents the common themes and topics of presentations at CNI member meetings from 2015-2019. Data were gathered from program PDFs, CNI meeting roadmap summaries, and other details for each individual meeting, which are published openly on CNI's website.

More details on the meeting programs can be found at [CNI Past Membership Meetings](https://www.cni.org/events/membership-meetings/past-meetings).

This visualization was developed by pairing with Claude AI to develop html to represent the shared themes and topics per CNI meeting, and is presented for exploration. It can be further developed with your feedback and suggestions. Please share those thoughts - and report errors - as pull requests!
